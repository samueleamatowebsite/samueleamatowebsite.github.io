Software I use

Operating System/Distribution
I use Void Linux, which is a minimalist distribution that utilizes runit for its init system and xbps as its package manager. You can read my opinion on Void Linux here.
Terminal
I use Alacritty, a fast, GPU-accelerated terminal emulator known for its simplicity and high performance. Its configuration is straightforward, making customization easy.
Window Manager/Desktop Environment
dwm from suckless.org, a super-fast and minimalist window manager. For my status bar, I use dwmblocks. To run my builds of dwm and dwmblocks, you'll need to install libxft-bgra until the patch is merged upstream.
Text editing and programming
Neovim serves as my text editor due to its high customizability, lightweight design, and excellent plugin support. It boosts productivity with features like asynchronous editing and a modern interface.
Web Browsing
I use Firefox as my web browser, enhanced with the user script BetterFox. This setup improves privacy and customizes my browsing experience.
